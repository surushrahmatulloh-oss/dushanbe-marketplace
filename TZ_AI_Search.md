# ТЗ Иловагӣ: Ҷустуҷӯ ва Ёрдамчии Зеҳни Сунъӣ (AI Search Assistant)
## Барои лоиҳаи "Душанбе Маркетплейс"

**Версия:** 1.0
**Мақсад:** Истифодабаранда бо забони одӣ (тоҷикӣ) менависад, ки чӣ мехоҳад — масалан "ба ман мошини арзон дар доираи 5000 сомонӣ лозим аст" ё "хонаи 2-хонагӣ наздики марказ" — ва зеҳни сунъӣ маҳсулоти мувофиқро аз байни ҳазорон эълон ёфта, нишон медиҳад.

Ин боби иловагӣ ба `TZ.md` ва `TZ_Feature_Jahoni.md` мебошад.

---

## 1. ИДЕЯИ АСОСӢ

Ба ҷои қуттии ҷустуҷӯи оддӣ (ки танҳо калимаҳоро мутобиқ мекунад), истифодабаранда метавонад:

- Бо ҷумлаи табиӣ бинависад: *"Барои фарзандам либоси зимистонӣ, на он қадар қимат"*
- Овоз (ихтиёрӣ, дар оянда): бо микрофон гап занад
- Ва зеҳни сунъӣ фаҳмад: категория (либоси кӯдакона) + фасл (зимистонӣ) + диапазони нарх (арзон) — баъд аз базаи маълумот беҳтарин мутобиқро ёбад.

Ин "AI Search Bar" метавонад дар саҳифаи асосӣ ба ҷои/дар паҳлуи ҷустуҷӯи оддӣ ҷойгир шавад.

---

## 2. ЧӢ ГУНА КОР МЕКУНАД (Архитектура)

```
Истифодабаранда матн менависад
        ↓
Frontend матнро ба API мефиристад (/api/ai-search)
        ↓
Backend ба Claude API (ё дигар LLM) дархост мефиристад
бо system prompt: "Аз ин матн параметрҳои ҷустуҷӯро 
бароварда, дар формати JSON бидеҳ"
        ↓
LLM бармегардонад: 
{ category: "kids-clothing", maxPrice: 200, 
  keywords: ["зимистонӣ", "либос"], season: "winter" }
        ↓
Backend бо ин параметрҳо ба PostgreSQL/Prisma дархост 
месозад (WHERE category = ... AND price <= ...)
        ↓
Натиҷаҳо ба истифодабаранда бо тавзеҳи АI нишон дода мешаванд:
"Ман 12 либоси зимистонии кӯдакона ёфтам то 200 сомонӣ"
```

## 3. МИСОЛИ ТЕХНИКӢ (API Route дар Next.js)

```typescript
// app/api/ai-search/route.ts
import { NextRequest, NextResponse } from "next/server";

export async function POST(req: NextRequest) {
  const { query } = await req.json();

  // Қадам 1: Фиристодани матн ба Claude барои таҳлил
  const aiResponse = await fetch("https://api.anthropic.com/v1/messages", {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "x-api-key": process.env.ANTHROPIC_API_KEY!,
      "anthropic-version": "2023-06-01",
    },
    body: JSON.stringify({
      model: "claude-sonnet-4-6",
      max_tokens: 500,
      system: `Ту ёрдамчии ҷустуҷӯи маркетплейс ҳастӣ. 
        Аз матни истифодабаранда параметрҳои ҷустуҷӯро барор.
        ТАНҲО JSON бидеҳ, чизи дигар нанавис:
        { "category": string|null, "minPrice": number|null, 
          "maxPrice": number|null, "keywords": string[], 
          "location": string|null }`,
      messages: [{ role: "user", content: query }],
    }),
  });

  const data = await aiResponse.json();
  const parsed = JSON.parse(data.content[0].text);

  // Қадам 2: Ҷустуҷӯ дар база бо параметрҳои бадастомада
  // (ин ҷо кодии Prisma барои филтр илова мешавад)

  return NextResponse.json({ filters: parsed });
}
```

> **Эзоҳ:** Ин мисоли соддашуда аст. Дар лоиҳаи воқеӣ бояд categories-и мавҷударо ба system prompt диҳед, то LLM танҳо категорияҳои воқеиро баргардонад (на ихтироъ кунад).

## 4. ХУСУСИЯТҲОИ ИЛОВАГӢ (Марҳилаи 2 ва минбаъд)

### 4.1 Тавзеҳи натиҷа бо забони табиӣ
Ба ҷои рӯйхати хушк, AI метавонад бинависад: *"Ман 8 мошин ёфтам, аммо 3-тои онҳо аз соли 2015 поёнтаранд — оё мехоҳед онҳоро низ бинам?"*

### 4.2 Ҷустуҷӯи бо расм (Visual Search)
Истифодабаранда расми чизеро бор мекунад (масалан куртаи дӯсташ), AI монандашро дар маркетплейс меёбад.
- Технология: CLIP embeddings ё Claude vision API барои тавсифи расм → баъд ҷустуҷӯи матнӣ дар асоси тавсиф

### 4.3 Чат-ёрдамчии харид (Shopping Assistant Chat)
Диалоги пай дар пай:
- Истифодабаранда: "Ба ман телефон лозим"
- AI: "Буҷаи шумо чӣ қадар аст ва барои чӣ мақсад — бозӣ, акс ё оддӣ?"
- Истифодабаранда ҷавоб медиҳад → AI натиҷаи мушаххас пешниҳод мекунад

### 4.4 Тавсифи автоматии эълон (барои фурӯшандагон)
Фурӯшанда танҳо расми маҳсулот бор мекунад → AI сарлавҳа ва тавсифи хуб пешниҳод мекунад (сарфаи вақт барои фурӯшанда).

### 4.5 Огоҳии нархи бозор (Price Intelligence)
AI нархи миёнаи бозорро барои маҳсулоти монанд таҳлил карда, ба фурӯшанда мегӯяд: "Нархи шумо аз миёнаи бозор 20% болотар аст."

## 6. МУЛОҲИЗАҲОИ МУҲИМ

- **Хароҷот**: Ҳар дархост ба Claude API арзиш дорад (аммо хеле кам — тахминан $0.001-0.003 барои ҳар ҷустуҷӯ бо Claude Haiku). Барои ин вазифаи содда истифодаи **Claude Haiku 4.5** тавсия мешавад, на модели калон — зуд ва арзон.
- **Кэшкунӣ**: Дархостҳои такрории умумӣ (масалан "телефон арзон") метавонанд кэш карда шаванд, то арзиш кам шавад.
- **Fallback**: Агар AI дастрас набошад ё хато диҳад, система бояд ба ҷустуҷӯи оддии калимавӣ баргардад (набояд сайт вайрон шавад).
- **Забон**: System prompt бояд ба таври возеҳ дар бораи забони тоҷикӣ ва хусусиятҳои маҳаллӣ (номи шаҳрҳо, воҳиди пул — сомонӣ) огоҳ карда шавад.

---

## 6.1 ХУСУСИЯТҲОИ UI-И ҶУСТУҶӮИ ҲУШМАНД (қарори қабулшуда)

- **Шакл**: Тугма/навори "Ҷустуҷӯи ҳушманд" пурра гирд (pill/capsule shape, border-radius: full), бо соя нарм.
- **Sticky**: Навор ҳангоми scroll кардани саҳифа ҳамеша намоён мемонад (боло дар десктоп, поён дар мобил).
- **Ҷустуҷӯи овозӣ (Voice Search)**: Иконаи микрофон дар паҳлуи тугма.
  - Технология: Web Speech API (`SpeechRecognition`), зеро ин ройгон ва built-in дар browser аст.
  - **Забонҳо**: тоҷикӣ дар Web Speech API дастгирӣ намешавад, бинобар ин ду забон интихоб шуд: **русӣ (`ru-RU`)** ва **англисӣ (`en-US`)**, бо тугмаи хурди тағйири забон (RU/EN) дар паҳлуи иконаи микрофон.
  - Ҳангоми сухан гуфтан — анимацияи pulse ("гӯш карда истодаам").
  - Матни бадастомада (аз ҳар ду забон) ба ҳамон `/api/ai-search` фиристода мешавад — Claude метавонад матни русӣ/англисиро фаҳмида, натиҷаро аз базаи тоҷикӣ ёбад (system prompt бояд ин нуктаро дар назар гирад: "матни вуруд метавонад ба забони русӣ ё англисӣ бошад, аммо натиҷаро аз маҳсулоти тоҷикӣ ёфта деҳ").
  - Агар browser Speech API-ро дастгирӣ накунад — иконаи микрофон пинҳон мешавад (graceful fallback).
